Web Application Usage Guide Lines

Initial Steps

  • Use Firefox browser version 5.1 or above .
  • Make sure your computer display resolution is at least 1280 X 1024 .
  • Create an account in the online web application www.adwiise.com  or call 312 343 0073 . The "Create Account" button can be found on the top right corner of the home page.
  • Buy the Android Media Player from us [Should be Android OS 2.3 or above versions] .
  • The online account will be activated for you by us.
  • logon to the online application
  • Upload media files to you account using the web/desktop client application
  • Make sure there is calendar entries whose end dates are greater than the current date
  • click the create daily calendar button on the calendar page.
  • Let us know in advance the start and end time for the app to be played every day.

Set Up WiFi

  • On your device go to settings-->wireless and networks--> manage and set up a WiFi connection.
  • Enable - Airplane Mode/Wi Fi
  • On your device go to settings-->wireless and networks--> enable Airplane Mode --> Enable Wi Fi

                     settings-->wireless and networks-->  Enable Wi Fi

  •  Set the time in 24 hour format-
  •   settings-->wireless and networks--> Date & Time -> Use 24 Hr Format

App Installation

  •  Create a directory in the device called adwiise_installables [ Internal Phone Storage] memory drive.
  •  Copy the first adwiise_signage_base  apk  [Base/Admin app] files received from adwiise into the above directory.
  •  Install the   file by clicking on the above apk file
  •  once the above app is  up and running , press the Device Machine Id button.
  •  Send us the value appearing on the dialog window.
  • Once we confirm the activation do the steps below
  •  Click the ok button on the window.
  •  Click the ‘Init App’ button . [ Wait for the process to be completed].
  •  Click the ‘Set Up’ Files button . [ Wait for the process to be completed].
  •  Click the ‘Contents’ button . [ Wait for the process to be completed].
  •  Copy the second  adwiise_signage_player  apk files received from adwiise into the above    directory.
  •  Install the   file by clicking on the above apk file
  •  Make sure the power chord is connected and the device is charging so that the screen won't go blank   or sleep mode.
  •  Click on the Adwiise Signage Base app
  •  Click the Start Service button  
  •  [optional] connect the HDMI cable to a TV/display and select Mirror on display.
  • [make sure to use files of the formats jpeg, png and mp4 videos in an Android device . All other formats may fail or will not run on the device.]
  •   Keep the phone/device in the landscape mode.
  •  Now Click on the Adwiise Signage Player app . The app should be up and running successfully.
